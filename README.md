Gesture Galaxy (개인 프로젝트)

프로젝트 소개 : 웹캠을 통해 사용자의 손동작을 실시간으로 인식하고, 화면 속 우주 파티클을 제어하는 인터랙티브 웹 디자인 프로젝트입니다.

기술 스택: HTML5 (Canvas API), CSS3, js, Google MediaPipe Hands, Firebase Hosting

기술 도입 배경 및 성과 :
JavaScript 및 Canvas API 활용: 무거운 프레임워크(React, Vue 등)의 오버헤드를 배제하고, 하드웨어 가속을 활용해 수백 개의 파티클을 60FPS로 끊김 없이 렌더링하도록 최적화했습니다.

MediaPipe Hands : 서버 통신 없이 클라이언트 브라우저 단에서 지연 시간(Latency)을 최소화하여 즉각적인 반응성을 확보했습니다.

Firebase Hosting : 누구나 링크 클릭 한 번으로 쉽게 프로젝트를 체험할 수 있도록 접근성을 높이고 서버 인프라를 구축했습니다.

주요 구현 기능 :
글래스모피즘(Glassmorphism) 기반의 직관적인 랜딩 페이지와 사용 안내 UI 구현.

손가락 관절의 $x, y$ 좌표를 수학적으로 계산하여, 손의 크기나 카메라 각도에 구애받지 않고 '주먹 쥐기'와 '손 펴기'를 정밀하게 판별하는 알고리즘 적용.

손동작 상태에 따라 파티클이 모여드는 인력(블랙홀)과 흩어지는 척력(빅뱅)을 구현한 커스텀 물리 엔진 적용.

추후 구현 기능 :
기존에 있는 손가락인식 기능이 인식이 잘안되기에 인식률을 높이는 방법을 찾아 보완할 것입니다.

전반적으로 퀄리티가 낮기에 조금씩 사용자의 사양에 맞춰 퀄리티를 올릴 것입니다.
